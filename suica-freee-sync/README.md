# Suica → freee 同期プラグイン

モバイルSuicaの利用履歴をfreee会計に差分同期するプラグインです。freee APIの自動同期が停止した場合の手動リカバリに使用します。

## 機能

- freeeの既存明細とSuica履歴を明細レベルで突合
- freeeに存在しない明細のみをCSVで作成・インポート
- 同一日・同一金額の重複にも対応（カウントベース照合）
- アップロード後の再照合で差分ゼロを検証

## コンポーネント

| 種類 | 名前 | 説明 |
|------|------|------|
| コマンド | `/suica-sync` | Suica→freee差分同期を実行 |
| スキル | `suica-freee-sync` | 同期ワークフローの詳細手順・スクリプト |

## 使い方

以下のいずれかで起動できます:

- `/suica-sync` コマンドを実行
- 「Suicaの明細をfreeeに同期して」と自然言語で依頼

## 前提条件

- Claude in Chrome（ブラウザ自動操作）が有効であること
- モバイルSuicaアカウントにログイン可能であること
- freee会計にモバイルSuica口座が登録済みであること
- Python 3 が利用可能であること

## 同期の仕組み

1. freeeの明細一覧から全Suica明細を抽出（日付・金額・内容）
2. Suicaサイトで利用履歴を取得
3. 明細レベルで差分照合（日付+金額+内容のタプルで突合）
4. freeeに存在しない明細のみCSVを作成
5. freeeにCSVアップロード
6. 再照合で差分ゼロを検証

# X Viral Repost Plugin

X（Twitter）でフォロー中ユーザーのバズ投稿を見つけ、人間らしいコメント付きで引用リポストするプラグイン。

## コンポーネント

| 種類 | 名前 | 説明 |
|------|------|------|
| コマンド | `/x-repost` | バズ投稿探し→引用リポストのワークフローを起動 |
| スキル | x-viral-repost | 投稿選定・コメント生成・ブラウザ操作のガイドライン |

## 使い方

- `/x-repost` と入力するだけで、自動的にXを開いてバズ候補を探し始めます
- 「Xでバズ投稿をリポストして」のような自然言語でもスキルが起動します

## 必要な環境

- Claude in Chrome（ブラウザ操作に必要）
- X（Twitter）にログイン済みのブラウザ

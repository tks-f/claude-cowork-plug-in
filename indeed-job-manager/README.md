# Indeed Job Manager プラグイン

Indeed求人の管理を自動化するCoworkプラグイン。

## スキル

### 1. indeed-job-analytics（求人分析）
Indeed Employer Dashboardから全求人のパフォーマンスデータ（表示回数・クリック数・応募数・CTR）を収集し、改善優先度をスコアリング。さらにIndeed検索で競合求人の給与相場・キーワードを調査し、自社求人との差分を分析する。結果は6シート構成のExcelレポートとして出力。

**トリガー例**: 「Indeedの求人を分析して」「求人のパフォーマンスを見せて」「競合の求人と比較して」「Indeed分析レポートを作って」

**出力内容**:
- サマリー（KPI一覧・優先度スコア）
- 競合分析（給与比較・市場ポジション）
- アクションプラン（優先度順）
- 改善コンテンツ（タイトル・給与・仕事内容・アピール・求める人材の全文改善案）

### 2. indeed-job-refresh（求人リフレッシュ）
掲載から5日以上経過したIndeed求人を自動的に終了→複製→無料再掲載する。再掲載によりIndeedの検索アルゴリズムをリセットし、表示機会を改善する。

**トリガー例**: 「Indeedの求人をリフレッシュして」「古い求人を再掲載して」

### 3. indeed-job-sync（求人同期）
Indeedの募集中求人を求人ボックス（採用ボード）とengage（エンゲージ）に同期する。各プラットフォームのSEO特性に合わせて求人内容を最適化して掲載する。

**トリガー例**: 「求人を同期して」「Indeedの求人を他媒体に展開して」

**同期内容**:
- Indeedにあり対象サイトにない求人 → 新規追加
- 両方にある求人 → 内容を比較し差分更新
- Indeedにない求人 → 対象サイトから削除

## 前提条件

- Claude in Chrome（ブラウザ自動操作）が利用可能
- 以下にログイン済み：
  - Indeed Employer Dashboard（employers.indeed.com）
  - 求人ボックス 採用ボード（求人ボックス.com）※同期スキル使用時
  - engage（en-gage.net）※同期スキル使用時
- 分析スキルはxlsxスキルを併用してExcelレポートを生成

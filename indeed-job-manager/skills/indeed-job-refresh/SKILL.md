# Indeed 求人リフレッシュスキル

## 概要

Indeed Employer Dashboard上で、掲載から5日以上経過した求人を終了→複製→無料再掲載するスキル。
再掲載によってIndeedのアルゴリズムにリセットをかけ、検索結果への表示機会を改善する効果がある。

## リフレッシュ対象

**掲載日から5日以上経過** した「募集中」の求人すべてが対象。

## 前提条件

- Claude in Chrome（ブラウザ自動操作）が利用可能であること
- ユーザーがIndeed Employer Dashboardにログイン済みであること
- 操作対象はemployers.indeed.comの日本語UI

## ワークフロー

### Step 1: 求人一覧の確認

1. `employers.indeed.com` にアクセス
2. 左メニューの「求人」をクリックして求人一覧を表示
3. ステータスフィルターが「募集中・休止中」であることを確認
4. 各求人の **求人タイトル** と **掲載日** を収集する

### Step 2: 対象求人の特定とユーザー確認

掲載日から5日以上経過した求人を特定する。
- 「X日前」表示の場合：Xが5以上なら対象
- 日付表示の場合：今日との差分が5日以上なら対象

対象が見つかったら、ユーザーに一覧を報告して確認を取る：

```
以下の求人がリフレッシュ対象です：
1. [求人タイトル] - 掲載日: YYYY/MM/DD
2. [求人タイトル] - 掲載日: YYYY/MM/DD

これらの求人を終了→複製→再掲載してよろしいですか？
```

対象がなければ「リフレッシュ対象の求人はありませんでした」と報告して終了。

### Step 3: 求人の終了

対象求人を1件ずつ終了する：

1. 求人一覧で「● 募集中」のステータスドロップダウンをクリック
2. 「募集終了」を選択
3. 終了理由ダイアログ：
   - Step 1: 「誰も採用しなかった」を選択
   - Step 2: 「候補者が見つからなかった」を選択
4. ステータスが「募集終了」に変わったことを確認

### Step 4: 求人の複製

1. 求人一覧のフィルターに「募集終了」を追加
2. 求人タイトルの一部で検索して対象を見つける
3. 求人行の右端「...」（三点メニュー）をクリック
4. 「複製する」を選択

三点メニューからの操作が最も確実。求人詳細ページの「アクション」ボタンはドロップダウンが不安定なため避ける。

### Step 5: 求人作成フローの通過

内容は変更せず、各ステップで「続ける」をクリックして進む：

1. 基本情報 → 「続ける」
2. 求人の詳細 → 「続ける」
3. 求人の募集要項 → 「続ける」
4. 応募関連の設定 → 「続ける」
5. 確認ページが表示される

### Step 6: 無料で掲載

1. 確認ページ最下部の「この内容で確定 →」をクリック
2. 有料プラン選択ページに遷移
3. ページ最下部の **「求人を無料掲載する」** リンクをクリック（「保存して次へ」は有料プランなので押さない）
4. 「求人の掲載設定が完了しました」が表示されれば成功

### Step 7: 繰り返しと完了報告

複数求人がある場合はStep 3〜6を繰り返す。全件完了後：

```
Indeed求人リフレッシュが完了しました。

処理結果：
- [求人タイトル1]: 終了 → 複製 → 再掲載完了（審査待ち）
- [求人タイトル2]: 終了 → 複製 → 再掲載完了（審査待ち）

※審査完了後に掲載されます（通常数時間）。メールで通知が届きます。
```

## UI操作の注意点

- **ドロップダウン**: メニュー項目の中央を正確にクリックする。マウスがメニュー外に出ると閉じる
- **ページ遷移**: ボタンクリック後は2〜3秒待ってからスクリーンショットを撮る
- **フィルター**: 「募集終了」フィルター追加後、次の操作前にリセットが必要な場合がある
- **検索**: 求人タイトルの一部（職種名の最初の数文字）で検索すると確実に見つかる

## エラーハンドリング

- **ログインが必要**: ユーザーにログインを依頼する。パスワード入力はユーザー自身が行う
- **ページ読み込みエラー**: 3秒待ってリトライ。2回失敗でユーザーに報告
- **複製フローでエラー**: スクリーンショットで状況を確認し、ユーザーに報告
